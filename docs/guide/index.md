# Retentioneering

Retentioneering is a clickstream analysis ecosystem for product teams and data analysts who want to understand how users move through their product.

It consists of two parts that work together:

- [Python library](/docs/quick-start) — an open-source library for exploratory analysis in Jupyter notebooks, VS Code, and Google Colab. Use it to build funnels, visualize transition graphs, run segmentation, and cluster user paths directly from your event data.
- **Platform** — a collaborative web application for teams, coming soon.

Both tools share the same underlying analytical models, so insights developed in a notebook translate directly to the platform, and vice versa.

## Who is it for?

Retentioneering is built for teams that collect behavioral event data — user actions, page views, feature interactions — and want to go beyond simple funnels or aggregate dashboards. It is particularly useful when:

- You want to understand the full shape of user journeys, not just conversion rates
- You need to segment users by behavior and compare how different groups navigate your product
- You want to run analysis in a notebook with full programmatic control, then share results with non-technical stakeholders through the platform
